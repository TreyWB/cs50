Password.c Pseudocode

Objectives:
- Get user input
- Use check function to check user input against criteria
- If criteria is met, check function should return true

Implementation:
- Iterate through each character of user input
- Use bool values for criteria, if met change bool to true
- In check function, evaluate if all criteria bools are true before returning true
-----
- Use is?() function; if return 0 then mark corresponding bool as false
- ONLY ALLOW BOOL UPPERCASE, etc. to CHANGE WHEN FALSE

Criteria:
1. One uppercase letter
2. One lowercase letter
3. One number
4. One symbol
